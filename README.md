# pallass

## Development Set Up

### Client

Prerequisites:

- [Node.js](https://nodejs.org/en/) v20.17.0

To run dev server:

```bash
cd client
npm i
npm run dev
```

### Server

- [Go](https://go.dev/) v1.22.4+
- (optional) [Air](https://github.com/air-verse/air)

To run dev server:

```bash
cd server
go run .
```

To run live reload dev server using Air:

```bash
cd server
air
```
